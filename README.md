# Banner Image Finder

A web application that scrapes websites to find and extract banner images.

## Features
- Extract banner images from any website
- Detects hero banners, slider images, and header graphics
- Download images directly
- Clean, modern UI

## Installation

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
Open `frontend/index.html` in your browser

## Usage
1. Enter a website URL
2. Click "Find Banners"
3. View and download found banner images

## Tech Stack
- Backend: Node.js, Express, Cheerio
- Frontend: HTML, CSS, JavaScript
